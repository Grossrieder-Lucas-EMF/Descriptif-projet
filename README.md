## Nom du projet
Présentation du groupe de musique Arctic monkeys.

## Pourquoi ce thème 
Arctic monkey est un groupe de musique rock britanique créé en 2002 que j'apprécie depuis déja quelque années, je souhaiterais donc faire un petit site  pouvant répertorier des iinformations sur eux en donnant plus d'information sur leur passé et leur carrière.

## Quels seront les aspects du thème abordés dans le site
leurs musiques, leurs albums, qui ils sont, ce qu'ils font maintenant, les membres, les concerts, ce qui les a fait connaitre.

## Quel est le public cible ?
Les gens ne connaissant pas bien le groupe ou même ceux qui écoutent déja qui veulent juste avoir un peu plus d'infos sur le groupe en lui même.
